# _Callback and arrow function introduction in array_

<b>

```javascript
function isEven(element) {
    if (element % 2 === 0) {
       return true;
     }
  return false;
}

console.log(isEven(5));
```
OR

```javascript
function isEven(element) {
  return element % 2 === 0;
}

console.log(isEven(5));


```
